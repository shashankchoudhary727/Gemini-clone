üì¶ **Installation**

**Prerequisites**

*   Node.js (version 16+)
*   npm or yarn
*   A valid API key from Google Gemini

**Steps to Run the Project Locally**

1.  **Clone the Repository**

    ```bash
    git clone [https://github.com/shashankchoudhary/gemini-clone.git](https://github.com/shashankchoudhary/gemini-clone.git)
    cd gemini-clone
    ```

2.  **Install Dependencies**

    ```bash
    npm install
    ```

    or

    ```bash
    yarn install
    ```

3.  **Set Up Environment Variables**

    *   Create a `.env` file in the root directory.
    *   Add your Google API key:

        ```makefile
        VITE_API_KEY=your_google_api_key
        ```

    *   Save the file.

4.  **Run the Project**

    ```bash
    npm run dev
    ```

    or

    ```bash
    yarn dev
    ```

5.  **Access the Application**

    Open your browser and visit:

    ```
    http://localhost:5173
    ```
